## 신약개발 플랫폼 대시보드 (React)

이 레포지토리는 신약개발 플랫폼 대시보드 목업을 React + Vite로 구성한 프로토타입입니다.

### 개발환경 구동 매뉴얼

1. 의존성 설치

```bash
npm install
```

2. 개발 서버 실행

```bash
npm run dev
```

3. 프로덕션 빌드

```bash
npm run build
```

4. 빌드 미리보기

```bash
npm run preview
```

### 권장 환경

- Node.js 18 이상
- npm 9 이상
